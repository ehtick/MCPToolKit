# 🗺️ Roadmap

Our vision for the Azure Cosmos DB MCP Toolkit. This roadmap is a living document and subject to change based on community feedback.

---

## 📍 Current Release: v1.1.2 (June 2026)

### ✅ What's Here Now
- **Hybrid search tool** with Reciprocal Rank Fusion (RRF)
- **Multi-provider embedding support** (Azure AI Services, Foundry, OpenAI)
- **Enterprise security** (Entra ID, Managed Identity, RBAC)
- **Production-ready** hosting on Azure Container Apps
- **Web testing UI** for interactive testing
- **Python client example** for Microsoft Foundry

---

## 🚧 Q3 2026: Developer Experience & Community

### Features in Progress

#### 🧪 **Extended Test Suite**
- Performance benchmarks (latency, throughput)
- Security test cases
- Multi-region failover tests
- **Fuzz testing** for input validation and edge cases
- Status: **Planned**

---

#### 🔍 **Agent Retrieval Tools**
- New MCP tools for semantic retrieval
- Retrieval-Augmented Generation (RAG) support
- Multi-step retrieval workflows
- Status: **Planned**

**Why it matters:**
- Enable complex agent reasoning (search → analyze → answer)
- Better RAG patterns for LLM accuracy
- Support for knowledge graph queries

---

## 🎯 Q4 2026: Advanced Features

### Planned Features

#### ☁️ **Managed Experience (Remote Endpoint)**
- Fully hosted MCP server (no deployment needed)
- Microsoft-managed infrastructure
- Built-in monitoring & auto-scaling
- SAAS pricing model
- Status: **Backlog**

**Why it matters:**
- Zero infrastructure management
- Instant availability (minutes, not hours)
- Built-in compliance & security
- Perfect for teams without DevOps resources

## 📝 How To Influence the Roadmap

### Share Your Ideas

1. **Feature Request** → Open an [Issue](https://github.com/AzureCosmosDB/MCPToolKit/issues/new?labels=enhancement)
   - Describe your use case
   - Explain why it matters to you
   - Include examples

2. **Vote on Existing Features** → 👍 React to issues you care about
   - Most-reacted features get prioritized

3. **Contribute** → Build features yourself!
   - See [CONTRIBUTING.md](../CONTRIBUTING.md)
   - Start with good-first-issue labels

4. **Discuss on Discussions** → [GitHub Discussions](https://github.com/AzureCosmosDB/MCPToolKit/discussions)
   - Architecture decisions
   - Design feedback
   - Community stories
